PROFIL JURUSAN RPL

Profil Rpl

Rekayasa perangkat lunak adalah pengubahan perangkat lunak itu sendiri guna mengembangkan, memelihara, dan membangun kembali dengan menggunakan prinsip reakayasa untuk menghasilkan perangkat lunak yang dapat bekerja lebih efisien dan efektif untuk pengguna.

Secara khusus tujuan Program Keahlian Rekayasa Perangkat Lunak adalah membekali peserta didik dengan keterampilan, pengetahuan dan sikap agar kompeten :.

1. Mengerti tentang Konsep Algoritma Pemrograman dan Konsep Perangkat Lunak

2. Merancang Sistem Informasi untuk membangun suatu Software atau Aplikasi

3. Membangun Software Aplikasi Database Berbasis Desktop, Web dan Mobile

Tujuan

Tujuan Program Keahlian
Apa Saja Tujuan Program Keahlian RPL di SMK Tamansiswa 2 Jakarta...?
  1. Mecetak lulusan yang berkualitas dan mampu bersaing di pasar tenaga kerja baik nasional, regional maupun internasional dibidang Teknologi Informatika – Rekayasa Perangkat Lunak (Software Engineering).
  2. Membekali peserta didik dengan keterampilan, pengetahuan dan sikap agar kompeten.
  3. Menginstalasi software dan mengoperasikan software aplikasi spesifik.
  4. Mengimplementasikan aplikasi-aplikasi, pemrograman berbasis client server, desktop,pengembangan web dll.
  5. Mengelola usaha di bidang pembuatan software aplikasi.
  6. Membekali peserta didik dengan Ilmu Pengetahuan, Teknologi, dan Seni agar mampu mengembangkan diri baik secara mandiri maupun melalui jenjang pendidikan yang lebih tinggi.

Kompetensi

kompetensi Lulusan RPL
Untuk Kompetensi Lulusan RPL di SMK Tamansiswa 2 Jakarta itu apa saja...?
  1. Mampu menginstalasi sistem operasi dan menginstalasi aplikasi-aplikasi komputer baik opened source ataupun closed source.
  2. Mampu merancang, membuat dan mengaplikasikan tampilan website secara statis.
  3. Mampu membuat dan mengaaplikasikan aplikasi berbasi android.
  4. Mampu membuat, mengelola dan memelihara aplikasi website dinamis.
  5. Mampu membuat, mengelola dan memelihara aplikasi berbasis desktop client-server.
  6. Mampu merancang, membuat, mengelola dan memelihara basis data client-server.
  7. Mampu merancang berbagai perangkat lunak berbagai platform dengan berbagai teknik pemodelan perangkat lunak.
